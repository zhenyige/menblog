## 📌 写在最前面

本手册汇总了一套训练 **想象力**（跨域连接）与 **判断力**（风险与收益权衡）的 **结构化方法论**。把它当做「思维操作系统」，而非知识大全。

**适用人群**
- 想让认知杯子变大但又不想浑浑噩噩的打工人
- 正在跨界转型，需要快速吸收新信息的人
- 希望在 AI、商业、人文中随时切换思维视角的通才

**使用方式**
用 Obsidian、Logseq、Notion、或者纯文本均可。建立四个文件夹作为「存储-发酵-反馈」的基本骨架：

```
📁 /sense        结构化采集的原料库
📁 /reflection   思维工具卡与每日复盘
📁 /archives/weekly   手写周更「思维重写」
📁 /blackbox      同伴批判与反馈沉淀
```

每周抽 2–3 小时即可把这套机制长成第二大脑，而非一次性读完就束之高阁。

## 🧠 Part 1：给大脑供给高质量信息原料

想象力的原料分为 **三类**：

| 类别 | 来源示例 | 质量要求 | 工具方法 |
|------|---------|---------|---------|
| **阅读** | 30–50 页/天 | 必须做笔记金句漏斗 | Obsidian 卡片 + 文末 12 字洞察 |
| **视听碎片** | 10 分钟以内一条视频 | 必须有逐字稿/关键短语 | 边看边手写 3–5 句 Key |
| **实体/虚拟体验** | 开会、旅行、VR 漫游 | 记 3 条意识流 | Merge 到 sense 中 |

### 🔧 立即动手做

1. 建立 `/sense` 文件夹
2. 下载一个 **高信噪比 RSS 源监控工具**，推荐：
   - [Miniflux](https://miniflux.app)（自托管，开源）
   - [Feedly](https://feedly.com) + [Readwise Reader](https://readwise.io/read)（云端方案）
3. 订阅 **至少 4 个深度源**：
   - Wait But Why（科普 + 想象力）
   - MIT Technology Review Deep Tech / Business
   - Ben's Bites（AI 行业早讯汇总）
   - 一个中文深度专栏（例如《得到》或《奇绩创坛》）
4. 每天用 **「一句话情绪」+「一个反直觉洞察」** 结束卡片，放入 `/sense/daily/`，以年月日作文件名；
   - 例如：`2026-04-28-San-Francisco-Notes.md`

## 🤖 Part 2：装上「认知工具卡」

- **想象力** = 跨域同构映射能力
- **判断力** = 类别化 + 概率 + 反馈环

把两套工具卡放到 `/reflection/` 里随身带：

### 🛠 想象力工具卡（Think Cards）

```markdown
# T01 —— 类比失效触发器
问：如果这个概念在 18 世纪会怎样？

记录模板：
- 当前视角：___
- 反直历史视角：___
- 新奇洞察：___
```

```markdown
# T02 —— 反事实母题
问：如果科技倒退 50 年，会出现 X 的替代方案吗？
用于：重新评估现代解决方案的非线性可能性
```

```markdown
# T03 —— 角色切换
问：我在 X L4 团队的预算突然降到 100 万，我现在的第一选择会是什么？
作用：多维约束下资源分层认知
```

### ⚖ 判断力工具卡（Judge Cards）

```markdown
# J01 —— 信息来源信任度矩阵

模板：
- 来源：[   ]
- 题目相关度：70% | 置信权重：50% | 是否可证伪：YES/NO
- 调整建议：___
```

```markdown
# J02 —— 套利空间索引

模板：
- 事件/产品：___
- 谁付出成本：___
- 谁坐享溢价：___
- 溢价可能长到哪里：___
```

```markdown
# J03 —— 反脆弱杠杆

模板（三条必填）：
- 黑天鹅概率 ≥ ? |
- 灰犀牛概率 ≥ ? |
- 最小可行性部署成本 ≤ 预期收益 × 0.??

---
当全部满足时 → OK；
否则即刻写「降低杠杆」方案。
```

> 📌 **工具卡的使用节奏**
> - 新信息当天完成 Txx + Jxx 的填写（≤8 条，约 10 分钟）
> - 同一条资料，至少出现 J01 + J03 各一条才可归档
> - **每周日晚整理一次**：把归档但未被 J03 及格的卡片挪回思考池子，48 小时重做

## ✍️ Part 3：让思维过程可交互、可复盘

### 3A 周更「思维重写」

**时间**：25 分钟/周（固定时间，一旦开始不许暂停）

**材料**：把 `/sense` 与 `/reflection` 内本周所有卡片 + 核心笔记拷贝到一张白纸或 A3 纸。**必须手写——禁止键盘录入。**

**流程**：

1. 25 分钟内，用自己的话 **重新排列** 本周看到的信息
2. 至少捏造 **1 个自己的概念**（不许照搬原文，可以是一个 KEYWORD、一个比喻、一张漫画草图）
3. 文末写 100–150 字「后验偏误」：
   - 哪块写得过于乐观？
   - 哪块忽略了长期杠杆？
   - 如果要重来 1 次，我会调整哪条假设？

**保存路径**：`/archives/weekly/YYYY-Www-Essay-Header.md`
- 例子：`2026-W17-Essay-AI-Economic-Logic.md`

### 3B 月度「批判性同伴」

挑 1–2 个思路相近的朋友：

- **Step 1**：把上月 Essay 发给对方
- **Step 2**：对方在 48h 内回 **三条**：
  - ① 哪处逻辑坍塌？
  - ② 哪处新信息被忽略？
  - ③ 一句话「盲点补充」
- **Step 3**：你只接受 ① + ② 的「反事实验证」；③ 记录到 `/blackbox/` 文件夹，文件名：`YYYY-MM-Blackbox-Companion-Label.md`
- **必填三栏**：

```markdown
- Source：___
- ourAction：___
- Reason：___ （为什么这次选择相信/怀疑）
```

> 🚨 **避坑提醒**
> - 不要找"应声虫"朋友
> - 对方能提出「超出你当前信念集」的反面证据才算合格

## 🧮 Part 4：培养周期的元认知锚点

把上面几条变成 **可量化的反思 reflex** 的可视化仪表盘：

| 锚点 / 指标 | 量化方法 | 目标值 | 复盘周期 |
|------------|---------|--------|---------|
| 想象力-想象度 | /reflection 内 Think Cards 新增 N /周 | ≥5 个概念创新/周 | 每周 |
| 判断力-准确率 | 给每个判断链打主观概率 → 对比后验事实 | ≥70% hits | 隔周 |
| 反脆弱杠杆-失效率 | /blackbox 里 J03 未达标的次数 | ≤15% | 月度 |

### 自动化辅助脚本（可选）

一个 Python 命令行小工具，用于批量抓取当周 `/sense`、`/reflection` 文件，计算 token 情绪倾向（简单版），可放进 cron 定时执行：

```python
# file: weekly_sentiment.py
import glob, os, json
from textblob import TextBlob

root = "/path/to/your/sense/weekly/*.md"
files = glob.glob(root)
total_pos = 0
files_cnt = 0

for fp in files:
    with open(fp, 'r', encoding='utf-8') as f:
        blob = TextBlob(f.read())
        score = blob.sentiment.polarity
        if score > 0.3:
            total_pos += 1
        files_cnt += 1

print(json.dumps({
    "weekly_files": files_cnt,
    "positive_ratio": round(total_pos / files_cnt, 3) if files_cnt else 0
}))
```

**执行**：

```bash
python3 ~/toolbox/wukong-cron/weekly_sentiment.py >> \
  ~/wukong/archives/weekly/sentiment_report.json
```

## 🧩 Part 5：7×21 天小闭环实验

用「50 天基线实验」来内化这套流程。不达标不轻言放弃，达标的人基本上都能把想象力/判断力升一个量级。

### Week 1–2
- 建起 `/sense`、`/reflection`、`/archives/weekly/`、`/blackbox/`
- 订阅 4–5 个高信噪比信息源，设置浏览器自动摘要到 `/sense/daily/`

### Week 3–6
- 每日 Think Cards + Judge Cards（≤8 条），选 1–2 个深度条目做周 +25 min 手写 Essay
- 周日晚：用 J03 打分，想象力 + 判断力各 ≥70 分为合格；否则下周继续练

### Week 7–8
- 找来 1–2 个批判性同伴
- 写第 1 版「长文评测」：
  - 字数：1500–2000
  - 题目：自拟，例如《AI + 人性 × 三年后的 10 种可能世界》
  - 评分员：同伴 或 你选择的 AI 模型
  - 打分维度：想象力×10 / 判断力×10 / 可读性×5 / 反思深度×5 → 满分 30 分

### Week 9–12
- 把前 8 周所有的评测分、主观概率 vs 后验命中率全量化表格化
- 调整自己的反脆弱杠杆系数（例如把 $k = 0.1 倍预期收益 → 0.07，降低认知损失）

### Week 13
- 写长文 + 让 AI 或同伴独立打分，公布两版对比，修订手册 v1.1

## ⚠️ 5 条隐含约束守则

1. **每天新建 ≤3 张思维卡片** → 周末前必须把所有灰色待整理卡片归零
2. **尽量不吃零食知识**：微博热搜、30 秒短视频、短剧情除非马上能用于反验证，否则直接屏蔽
3. **必须写在 markdown 文件里**；碎片日记禁止存在单个聊天软件里
4. **允许想象力溢出——但必须跟 J03 立刻对表**；不然变成空想
5. **一个月内至少要有一次让批判性同伴看到 Essay 并给出不可爱但可信的反馈**；如果找不到同伴，就写成公开文章发到社区（例如少数派、得到、GitHub）并主动接受路人评议

## 🧪 自测题：你的思维系统强度

每项 1–5 分，总分 20 分满。

| 题目 | 满分 |
|------|------|
| 我能把一条新闻在 5 分钟内拆解成 3 个维度，加上 1 个反直觉洞察？ | 5 |
| 我会把每个决策的主观概率用四分位打钉并记录后验命中/失手？ | 5 |
| 本周我读完且做了笔记的有效信息源 ≥10 条？ | 5 |
| 我已经写了至少 2 篇 1500 字以上的 Essay 复盘？ | 5 |
| 我至少找到 1 个批判性同伴进行过月度同行打分？ | 5 |

**总分：___ / 20**

- 15–20：恭喜，思维操作系统已成形
- 9–14：还有空白，继续练
- 0–8：重新阅读本手册 + 设 50 天实验闹钟
